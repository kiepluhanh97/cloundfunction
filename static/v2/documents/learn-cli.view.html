Setting up the Cloud Functions CLI

Start Creating

To use the Cloud Functions CLI (Command Line Interface), follow these steps.

1. Download

Download and install the OpenWhisk CLI.

2. Extract

Extract the downloaded file to OpenWhisk_CLI-latest-linux-amd64 folder

3. Configure Host and Authentication

Set API Host:

./wsk property set --apihost {{APIHOST}}

Set Authentication:
./wsk property set --auth API Key

4. Test It

Verify your setup. Here, we perform a blocking (synchronous) invocation of echo, passing it "hello" as an argument.

./wsk -i action invoke /whisk.system/utils/echo -p message hello --result

Result:

{ "message": "hello"}
